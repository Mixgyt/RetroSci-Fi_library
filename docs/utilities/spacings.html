Spacings

Son los elementos de espaciado basicos utilizados en html, pero ahora compactados y listos en clases predefinidas listas para su uso.

Margins y Paddings

Antes de empezar con los ejemplos una pequeña explicacion sobre la implementación de dichos espaciados basicos en la libreria.

La notacion que se ha utilizado es la misma que utiliza bootstrap asi que en unos partes para mayor sencillez es lo mismo.

Los margins predefinidos usan una medida estandar basada en 1rem las demas medidas siendo tambien basadas en la medida base tal como se muestra a continuación.

  • 0 - 0rem
  • 1 - 0.25rem
  • 2 - 0.50rem
  • 3 - 1rem
  • 4 - 1.5rem
  • 5 - 3rem
  • auto - auto

Cada una de las medidas anterior listadas aplican tanto para los margins como para los paddings.

Notación utilizada

Esta pequeña la dedicara para explicar un poco mas sobre la notacion utilizada para los spacings y sus nombres de clase respectivos para su utilización.

Margins y Paddings:

  • m - esto es la notacion para margin
  • p - esta es la notacion para padding

Notacion de las posiciones:

  • t - este seria el equivalente a colocar top, asi margin-top o padding-top
  • b - este seria el equivalente a colocar bottom, asi margin-bottom o padding-bottom
  • s - este seria el equivalente a colocar left, asi margin-left o padding-left y se usa "s" por la notacion start.
  • e - este seria el equivalente a colocar right, asi margin-right o padding-right y se usa "e" por la notacion end.
  • x - este seria el equivalente a colocar right y left a la vez, es decir aplicando el espaciado en el eje horizontal del elemento.
  • y - este seria el equivalente a colocar top y bottom a la vez, es decir aplicando el espaciado en el eje vertical del elemento.

Ejemplos de dichas clases

SCSS

$spacer: 1rem;

.mt-0 {
  margin-top: 0 !important;
}

.ms-1 {
  margin-left: ($spacer * .25) !important;
}
           
.px-2 {
  padding-left: ($spacer * .5) !important;
  padding-right: ($spacer * .5) !important;
}

.p-3 {
  padding: $spacer !important;
}

Centrado Horizontal (Propiedad auto)

Utilizar propiedades auto puede ser util en casos de querer centrar ciertos tipos de contenedores o para dejar que el html se adapte solo.

Centered element
HTML
  
<div class="mx-auto bg-light-subtle" style="width: 200px;">
  Centered element
</div>

Gap

Esta propiedad es util cuando usamos grid por que nos ayuda a dar un espaciado entre cada elemento dentro del grid, y los valores que .gap puede tomar van desde 0 a 5.

Veremos un ejemplo sencillo de como usarlo a continuación:

Grid item 1
Grid item 2
Grid item 3
HTML
  
<div class="grid gap-2">
  <div class="p-2 g-col-12 border">Grid item 1</div>
  <div class="p-2 g-col-12 border">Grid item 2</div>
  <div class="p-2 g-col-12 border">Grid item 3</div>
</div>
