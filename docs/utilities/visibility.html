Visibility

Estas son clases que utilizan para poder ocultar ciertos elementos de la pantalla o para mostrarlos. Son utiles para añadirse desde javascript y solamente desaparecer o aparecer elementos en el html.

Clases visibile e invisible

Estas son clases que tal como dicen sus nombres, hacen visibles o invisibles los elementos a los cuales se le colocan.

Elemento visible
Elemento invisible
Elemento visible 2
HTML
  
<div class="row gap-3 justify-content-center">
    <div class="visible border col-12">Elemento visible</div>
    <div class="invisible border col-12">Elemento invisible</div>
    <div class="visible border col-12">Elemento visible 2</div>
</div>

Como podemos observar cuando el elemento se le coloca .invisibility solo no se observa pero el espacio del mismo el documento sigue presente para solventar eso utilizamos otra clase si queremos que parezca en el html como que dicho elemento no exista.

Clase .visually-hidden

Esta clase hace que el elemento al que se le coloca se haga invisible y a la vez su presencia ya no altere el html visible.

Ahora volvemos al ejemplo anterior para poder visualizarlo mejor:

Elemento visible
Elemento invisible
Elemento visible 2
HTML
  
<div class="row gap-3 justify-content-center">
    <div class="visible border col-12">Elemento visible</div>
    <div class="visually-hidden border col-12">Elemento invisible</div>
    <div class="visible border col-12">Elemento visible 2</div>
</div>

Ahora que se le coloco la clase .visually-hidden podemos observar que el elemento parece a simple vista como que no existiera a pesar que se encuentra en el html y no distorciona la posicion de los demas componentes.

Opacity

Esta son un grupo de clases que ayudan si queremos que un elemento no sea completamente invisible y queremos que se vea un poco.

Los valores que opacity puede tomar son 0,25,50,75 y 100

Opacity-0
Opacity-25
Opacity-50
Opacity-75
Opacity-100
HTML
  
<div class="row gap-3 justify-content-center">
    <div class="opacity-0 border col-12">Opacity-0</div>
    <div class="opacity-25 border col-12">Opacity-25</div>
    <div class="opacity-50 border col-12">Opacity-50</div>
    <div class="opacity-75 border col-12">Opacity-75</div>
    <div class="opacity-100 border col-12">Opacity-100</div>
</div>

Esta clase funciona para cualquier componente o contenedor pero eso si, modifica toda la visualización de dicho elemento aplicanose así tambien a sus hijos.
