Card

Una CARD es un contenedor de contenido flexible y extensible. Incluye opciones para encabezados y pies de página, una amplia variedad de contenido, colores de fondo contextuales y potentes opciones de visualización.

Botones

Los badges se pueden usar como parte de enlaces o botones para proporcionar un contador.

HTML


<button type="button" class="btn btn-info">
    Notificaciones <span class="badge">4</span>
</button>

Resultado

Posicionado

Utilice utilidades para modificar un .badge y posicionarlo en la esquina de un enlace o botón.

HTML


<button type="button" class="btn btn-info position-relative">
    Bandeja de entrada
    <span class="position-absolute top-0 start-100 translate-middle badge rounded-pill bg-danger">
        99+
        <span class="visually-hidden">mensajes no leídos</span>
    </span>
</button>

Resultado

También puedes reemplazar la clase .badge con algunas utilidades más sin recuento para obtener un indicador más genérico.

HTML


<button type="button" class="btn btn-info position-relative">
    Perfil
    <span class="position-absolute top-0 start-100 translate-middle p-2 bg-danger border border-light rounded-circle">
        <span class="visually-hidden">Nuevas Alarmas</span>
    </span>
</button>

Resultado

Colores de fondo

Establezca un background-color para primer plano con contraste en color para el texto.

HTML


<span class="badge bg-primary">Primary</span>
<span class="badge bg-secondary">Secondary</span>
<span class="badge bg-success">Success</span>
<span class="badge bg-danger">Danger</span>
<span class="badge bg-warning">Warning</span>
<span class="badge bg-info">Info</span>
<span class="badge bg-light">Light</span>
<span class="badge bg-alert">Alert</span>

Resultado

Primary Secondary Success Danger Warning Info Light Alert

Redondeado en Badges

Utilice la clase .rounded-pill de utilidad para hacer que los badges sean más redondeadas con un border-radius.

HTML


<span class="badge rounded-pill bg-primary">Primary</span>
<span class="badge rounded-pill bg-secondary">Secondary</span>
<span class="badge rounded-pill bg-success">Success</span>
<span class="badge rounded-pill bg-danger">Danger</span>
<span class="badge rounded-pill bg-warning">Warning</span>
<span class="badge rounded-pill bg-info">Info</span>

<span class="badge rounded-pill bg-alert">Alert</span>
