FORMULARIOS

Los formularios permiten la entrada y envío de datos mediante campos interactivos. Esta sección incluye controles de texto, selectores, botones, checks y estilos personalizados que se adaptan a múltiples escenarios de uso.

FORMULARO SIMPLE

Un formulario básico con campos de entrada de texto, contraseña y casillas de verificación. Este tipo de estructura es ideal para recopilación de datos en interfaces simples y accesibles.

HTML
  
<form>
    <div class="mb-3">
        <label for="exampleInputEmail1" class="form-label">Email address</label>
        <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp">
        <div id="emailHelp" class="form-text">
            We'll never share your email with anyone else.
        </div>
    </div>

    <div class="mb-3">
        <label for="exampleInputPassword1" class="form-label">Password</label>
        <input type="password" class="form-control" id="exampleInputPassword1">
    </div>

    <div class="mb-3 form-check">
        <input type="checkbox" class="form-check-input" id="exampleCheck1">
        <label class="form-check-label" for="exampleCheck1">Check me out</label>
    </div>

    <button type="submit" class="btn btn-primary">Submit</button>
</form>

Datalists

Los datalists permiten crear campos de texto con autocompletado usando una lista de opciones predefinidas. Son útiles para ayudar al usuario a escribir datos correctos sin limitarlo completamente como un menú select.

HTML
  
<label for="exampleDataList" class="form-label">De donde eres</label>
<input class="form-control" list="datalistOptions" id="exampleDataList" placeholder="Type to search...">
<datalist id="datalistOptions">
    <option value="San Francisco">
    <option value="New York">
    <option value="Seattle">
    <option value="Los Angeles">
    <option value="Chicago">
</datalist>

Form Select

El elemento <select> permite al usuario elegir una opción de una lista desplegable. Es útil cuando se necesita limitar la entrada a un conjunto definido de valores preestablecidos.

HTML
 
<label for="">Escoja un numero </label> 
<select class="form-select" aria-label="Default select example">
    <option selected>Open this select menu</option>
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
</select>

Select multiple

Este tipo de select permite al usuario elegir múltiples opciones al mismo tiempo. Para seleccionar varios elementos, se utiliza Cmd + clic en macOS o Ctrl + clic en Windows.

HTML
  
<select class="form-select" multiple aria-label="Multiple select example">
    <option selected>Open this select menu</option>
    <option value="1">One</option>
    <option value="2">Two</option>
    <option value="3">Three</option>
</select>

Input group

Los input group permiten combinar campos de entrada con elementos adicionales como texto o iconos. Se usan para mejorar la experiencia del usuario al mostrar un contexto claro, como el símbolo @ en un campo de usuario.

@
HTML
  
                              <div class="input-group flex-nowrap">
    <span class="input-group-text" id="addon-wrapping">@</span>
    <input type="text" class="form-control" placeholder="Username" aria-label="Username" aria-describedby="addon-wrapping">
</div>

Input group ejemplos

Los input group permiten extender campos de entrada añadiendo texto, botones o áreas de texto a los lados. Son útiles para mostrar formatos, símbolos o para construir campos compuestos como nombres de usuario, precios o URLs.

@
@example.com
https://example.com/users/
Example help text goes outside the input group.
$ .00
@
With textarea

    <input type="text" class="form-control" placeholder="Username" aria-label="Username" aria-describedby="basic-addon1">
</div>

<div class="input-group mb-3">

</div>

    <label for="basic-url" class="form-label">Your vanity URL</label>
    <div class="input-group">
        <span class="input-group-text" id="basic-addon3">https://example.com/users/</span>
        <input type="text" class="form-control" id="basic-url" aria-describedby="basic-addon3 basic-addon4">

    <div class="form-text" id="basic-addon4">Example help text goes outside the input group.</div>

<div class="input-group mb-3">

Input group con checkbox y radio

Podés integrar checkboxes o radios dentro de un input group para agregar opciones de selección directa junto a campos de texto. Esto es útil cuando se desea asociar rápidamente una acción o estado a la entrada.

HTML
  
<div class="input-group">
    <div class="input-group-text">
        <input class="form-check-input mt-0" type="checkbox" value="" aria-label="Checkbox for following text input">
    </div>
    <input type="text" class="form-control" aria-label="Text input with checkbox">
</div>

<div class="input-group">
    <div class="input-group-text">
        <input class="form-check-input mt-0" type="radio" value="" aria-label="Radio button for following text input">
    </div>
    <input type="text" class="form-control" aria-label="Text input with radio button">
</div>

Form grid

El sistema de grillas de Bootstrap permite alinear múltiples campos en una misma fila. Usando las clases row y col, podés organizar formularios de forma responsiva y ordenada.

HTML
  
<div class="row">
    <div class="col">
        <input type="text" class="form-control" placeholder="First name" aria-label="First name">
    </div>
    <div class="col">
        <input type="text" class="form-control" placeholder="Last name" aria-label="Last name">
    </div>
</div>
