Alerta Básica

Las alerts sirven para mostrar mensajes al usuario de forma destacada.

Alertas.crearAlerta({
        tipo: 'success',
        titulo: '¡Bienvenido!',
        mensaje: 'hola mundo',
});
HTML

<button class="btn btn-danger" onclick="Alertas.crearAlerta({tipo:'success', titulo:'Bienvenido', mensaje:'hola mundo'})">Alerta</button>

Tipos de Alertas

Existen 4 diferentes tipos de alertas que podemos aplicar cambiando la propiedad, cuando creamos la alerta:

Alertas.crearAlerta({
  tipo: 'success',
  tipo: 'info',
  tipo: 'error',
  tipo: 'warning'
});
HTML

<button class="btn btn-danger" onclick="Alertas.crearAlerta({tipo:'success', titulo:'Bienvenido', mensaje:'hola mundo'})">Alerta</button>

Parámetros de Alertas: Edición y Personalización

La función Alertas.crearAlerta permite personalizar las alertas con los siguientes parámetros:

Alertas.crearAlerta({
  tipo: 'success', // Tipo de alerta: 'success', 'info', 'error', 'warning'
  titulo: 'Éxito', // Título de la alerta
  mensaje: 'Los datos se han guardado correctamente.', // Mensaje de la alerta
  alert_neon: true, // Activa o desactiva el efecto de neón
  autoDesaparecer: true, // Define si la alerta desaparece automáticamente
  duracion: 4000, // Duración en milisegundos antes de desaparecer
  bg: 'bg-success', // Clase CSS para el fondo de la alerta
  text_color: 'text-white', // Clase CSS para el color del texto
  borde: 'border-success', // Clase CSS para el borde de la alerta
  estilo_brillo: true // Activa o desactiva el efecto de brillo
});
HTML

<button class="btn btn-success" onclick="Alertas.crearAlerta({tipo:'success', titulo:'Éxito', mensaje:'Los datos se han guardado correctamente.', alert_neon: true, autoDesaparecer: true, duracion: 4000, bg: 'bg-success', text_color: 'text-white', borde: 'border-success', estilo_brillo: true})">Mostrar Alerta</button>
