Navbar

El navbar es un componente esencial que permite la navegación entre las diferentes secciones de un sitio web. Generalmente se ubica en la parte superior e incluye enlaces, menús desplegables y botones que guían al usuario a través del contenido del sitio.

Marca

El .navbar-brand se puede aplicar a la mayoría de los elementos, pero un div funciona mejor, ya que algunos elementos pueden requerir clases de utilidad o estilos personalizados.

Navbar
HTML
  
          
<nav class="navbar bg-info">
    <div class="navbar-header">
        <div class="navbar-brand">Navbar</div>
    </div>
</nav>

Imagen

Puedes reemplazar el texto dentro de .navbar-brand con una clase dentro de un div.

HTML
  
          
<nav class="navbar bg-info">
    <div class="navbar-header">
        <div class="navbar-brand">
            <div class="img-logo-prin-white"></div>
        </div>
    </div>
</nav>

Imagen y texto

También puedes usar algunas utilidades adicionales para agregar una imagen y texto a la vez.

Navbar
HTML
  
          
<nav class="navbar bg-info">
    <div class="navbar-header">
        <div class="navbar-brand">
            <div class="img-logo-prin-white"></div>
            Navbar
        </div>
    </div>
</nav>

Navegación

Para poner crear un entorno de navegacion coloca la clase navbar-nav y el nav-item para colocar el diseño de los apartados.

Inicio Servicios Menu
HTML
  
          
<nav class="navbar bg-info" id="Navegacion">
    <button class="navbar-toggle" onclick="toggleMenu('Navegacion')">
        <div class="img-menu-white"></div>
    </button>
    <div class="navbar-nav">
        <a class="nav-item" href="#">Inicio</a>
        <a class="nav-item" href="#">Servicios</a>
        <a class="nav-item" href="#">Menu</a>
    </div>
</nav>

Formularios

Coloque varios controles de formulario y componentes dentro de una barra de navegación:

HTML
  
          
<nav class="navbar bg-info" id="Navegacion">
    <button class="navbar-toggle" onclick="toggleMenu('Navegacion')">
        <div class="img-menu-white"></div>
    </button>
    <div class="navbar-nav">
        <a class="nav-item" href="#">Inicio</a>
        <a class="nav-item" href="#">Servicios</a>
        <a class="nav-item" href="#">Menu</a>
    </div>
</nav>

Con los formularios puedes crear mas estilos aplicando las clases que ya vimos.

Navbar
HTML
  
          
<nav class="navbar bg-info p-3" id="Formularios1">
    <div class="navbar-header">
        <div class="navbar-brand">
            Navbar
        </div>
        <button class="navbar-toggle" onclick="toggleMenu('Formularios1')">
            <div class="img-menu-white"></div>
        </button>
    </div>
    <div class="navbar-nav">
        <input type="text" class="form-control" placeholder="Buscar">
        <button class="btn btn-success" type="Buscar">Buscar</button>
    </div>
</nav>

Tambien podemos usar todo el navbar para un formulario.

@
HTML
  
          
<nav class="navbar bg-info p-3">
    <div class="input-group">
        <span class="input-group-text " id="basic-addon1">@</span>
        <input type="text" class="form-control" placeholder="Username" aria-label="Username" aria-describedby="basic-addon1">
    </div>
</nav>

Botones

El uso de botones no esta restringido solo a unos pocos si no a todos garcias a la forma responsiva del navbar.

Link
HTML
  
          
<nav class="navbar bg-secondary-neon p-3">
    <button class="btn btn-outline-primary btn-lg " role="button" aria-disabled="true">Button Large</button>
    <button class="btn btn-outline-success btn-sm " role="button" aria-disabled="true">Button small</button>
    <a class="btn btn-info disabled" role="button" aria-disabled="true">Link</a>
    <button type="button" class="btn btn-primary">Primary</button>
    <button type="button" class="btn btn-secondary">Secondary</button>
    <button type="button" class="btn btn-success">Success</button>
    <button type="button" class="btn btn-outline-info">Info</button>
    <button type="button" class="btn btn-warning">Warning</button>
    <button type="button" class="btn btn-danger">Danger</button>
    <button type="button" class="btn btn-light">Light</button>
    <button type="button" class="btn btn-dark">Dark</button>
    <button type="button" class="btn btn-link">Link</button>
</nav>

Menú Desplegable

Dentro de una clase .nav-item puede haber un menu desplegable lo cual es muy conveniente en casos de necesitar mas opciones en una sola opcion

Cuando la pantalla se hace pequeña se crea un menu de hamburguesa por la clase .navbar-toggle.

Retro Sci-Fi
Inicio Servicios
Menú
Opción 1 Opción 2
HTML
  
          
<nav class=" navbar bg-info" id="Desplegable">
    <div class="navbar-header">
        <div class="navbar-brand">
            <div class="img-logo-prin-white"></div>Retro Sci-Fi
        </div>
        <button class="navbar-toggle" onclick="toggleMenu('Desplegable')">
            <div class="img-menu-white"></div>
        </button>
    </div>
    <div class="navbar-nav">
        <a class="nav-item" href="#">Inicio</a>
        <a class="nav-item" href="#">Servicios</a>
        <div class="nav-item dropdown">
            Menú
            <div class="dropdown-menu">
                <a class="dropdown-item" href="#">Opción 1</a>
                <a class="dropdown-item" href="#">Opción 2</a>
            </div>
        </div>
    </div>
</nav>
