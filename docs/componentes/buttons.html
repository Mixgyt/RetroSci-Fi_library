Botones

Los botones son elementos clave en la interfaz para ejecutar acciones y navegar por la aplicación. Esta sección presenta estilos, tamaños, estados y variaciones personalizadas de botones que se adaptan al tema Retro Sci-Fi.

Botones

Los botones utilizan clases temáticas basadas en la paleta de colores del sistema. Cada variante representa un contexto diferente como acciones primarias, advertencias o estados informativos. Todos los estilos son personalizables mediante Sass y se adaptan a la estética Retro Sci-Fi.

HTML
  
    <button type="button" class="btn btn-primary">Primary</button>
    <button type="button" class="btn btn-secondary">Secondary</button>
    <button type="button" class="btn btn-success">Success</button>
    <button type="button" class="btn btn-info">Info</button>
    <button type="button" class="btn btn-warning">Warning</button>
    <button type="button" class="btn btn-danger">Danger</button>
    <button type="button" class="btn btn-light">Light</button>
    <button type="button" class="btn btn-dark">Dark</button>

Botones desactivados

Los botones pueden desactivarse usando el atributo disabled para prevenir la interacción del usuario. Los estilos desactivados conservan su apariencia temática, pero aplican opacidad y eliminan efectos hover para indicar visualmente que no están disponibles.

HTML
  
    <button type="button" class="btn btn-primary" disabled>Primary</button>
    <button type="button" class="btn btn-secondary" disabled>Secondary</button>
    <button type="button" class="btn btn-success" disabled>Success</button>
    <button type="button" class="btn btn-info" disabled>Info</button>
    <button type="button" class="btn btn-warning" disabled>Warning</button>
    <button type="button" class="btn btn-danger" disabled>Danger</button>
    <button type="button" class="btn btn-light" disabled>Light</button>
    <button type="button" class="btn btn-dark" disabled>Dark</button>

Tamaños

Los botones pueden adaptarse a diferentes tamaños utilizando clases utilitarias como .btn-lg para botones grandes y .btn-sm para botones pequeños. Esto permite mantener coherencia visual en distintos contextos dentro de la interfaz.

HTML
  
    <button type="button" class="btn btn-primary btn-lg">Large button</button>
    <button type="button" class="btn btn-primary">Default button</button>
    <button type="button" class="btn btn-primary btn-sm">Small button</button>

Botones con estilo Outline

Los botones Outline ofrecen una alternativa visual más ligera al estilo sólido. Conservan el color temático en el borde y el texto, y aplican un fondo al hacer hover. Son ideales para acciones secundarias o menos destacadas.

HTML
  
    <button type="button" class="btn btn-outline-primary">Primary</button>
    <button type="button" class="btn btn-outline-secondary">Secondary</button>
    <button type="button" class="btn btn-outline-success">Success</button>
    <button type="button" class="btn btn-outline-info">Info</button>
    <button type="button" class="btn btn-outline-warning">Warning</button>
    <button type="button" class="btn btn-outline-danger">Danger</button>
    <button type="button" class="btn btn-outline-light">Light</button>
    <button type="button" class="btn btn-outline-dark">Dark</button>

Botones con efecto de brillo

Los botones con la clase .estilo-brillo aplican un efecto de resplandor al pasar el cursor, generando un aura visual que destaca la interacción. Este estilo es ideal para interfaces llamativas o temas con estética retro-futurista.

HTML
  
    <button type="button" class="btn btn-primary estilo-brillo">Primary</button>
    <button type="button" class="btn btn-secondary estilo-brillo">Secondary</button>
    <button type="button" class="btn btn-success estilo-brillo">Success</button>
    <button type="button" class="btn btn-danger estilo-brillo">Danger</button>
    <button type="button" class="btn btn-warning estilo-brillo">Warning</button>
    <button type="button" class="btn btn-info estilo-brillo">Info</button>
    <button type="button" class="btn btn-light estilo-brillo">Light</button>
    <button type="button" class="btn btn-dark estilo-brillo">Dark</button>
