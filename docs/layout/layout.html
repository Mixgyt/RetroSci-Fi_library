Clase .row

La clase .row es el contenedor principal para agrupar columnas. Utiliza display: flex y flex-wrap: wrap para organizar las columnas en una fila.

      <div class="col">Columna 1</div>
      <div class="col">Columna 2</div>
      <div class="col">Columna 3</div>
</div>

Clase .row-cols-*

La clase .row-cols-* permite definir el número de columnas que debe tener una fila sin necesidad de especificar el ancho de cada columna individualmente. Es útil para crear diseños responsivos rápidamente.

También se pueden usar variantes responsivas como .row-cols-sm-*, .row-cols-md-*, etc., para ajustar el número de columnas según el tamaño de la pantalla.

    <div class="col">Columna 3</div>

</div>

Alineación en Filas

Las clases align-items-* y justify-content-* permiten alinear vertical u horizontalmente los elementos dentro de una fila.

Centro
Final
HTML

<div class="row align-items-center justify-content-between" style="height: 150px;">
  <div class="col-3 bg-primary text-white">Inicio</div>
  <div class="col-3 bg-success text-white">Centro</div>
  <div class="col-3 bg-danger text-white">Final</div>
</div>

Columnas de Tamaño Fijo

Autoajuste con .col-auto

La clase .col-auto ajusta el ancho de la columna al contenido, mientras que las demás columnas ocupan el espacio restante.

Ajustado
Resto
HTML

<div class="row">
  <div class="col-auto bg-warning text-dark">Ajustado</div>
  <div class="col bg-info text-white">Resto</div>
</div>

Autoajuste Responsivo

Con .col-{breakpoint}-auto podés aplicar autoajuste en determinados tamaños de pantalla.

Contenido
Flexible
HTML

<div class="row">
  <div class="col-auto col-md-auto bg-primary text-white">Contenido</div>
  <div class="col bg-secondary text-white">Flexible</div>
</div>

Clase .grid y .g-col-*

La clase .grid organiza elementos en un sistema de cuadrícula. Las clases .g-col-* definen cuántas columnas ocupa un elemento (de un total de 12).

Columna 1
Columna 2
Columna 3
HTML

<div class="grid">
  <div class="g-col-4 bg-primary text-white">Columna 1</div>
  <div class="g-col-4 bg-warning text-dark">Columna 2</div>
  <div class="g-col-4 bg-success text-white">Columna 3</div>
</div>

Filas en Grid: .g-row-{n}

Esta clase define cuántas filas debe ocupar un elemento en un .grid.

    <div class="g-col-6 g-row-2 bg-danger-neon text-white">2 filas</div>

Filas Responsivas

Define filas en distintos tamaños de pantalla con .g-row-sm-*, .g-row-md-*, etc.

Este elemento cambia de altura:

  • 1 fila en pantallas pequeñas
  • 2 filas en medianas
  • 3 filas en grandes
  • 4 filas en extra grandes
Fijo: 1 fila
Fijo: 2 filas
HTML
<div class="grid">
    <div class="g-col-12 g-row-sm-1 g-row-md-2 g-row-lg-3 g-row-xl-4 bg-success text-white text-center p-2">
      <p class="fw-bold">Este elemento cambia de altura:</p>
      <ul class="list-unstyled">
        <li>1 fila en <strong>pantallas pequeñas</strong></li>
        <li>2 filas en <strong>medianas</strong></li>
        <li>3 filas en <strong>grandes</strong></li>
        <li>4 filas en <strong>extra grandes</strong></li>
      </ul>
    </div>
    <div class="g-col-6 g-row-1 bg-warning text-dark text-center p-2">
      Fijo: 1 fila
    </div>
    <div class="g-col-6 g-row-2 bg-info text-white text-center p-2">
      Fijo: 2 filas
    </div>
  </div>

Filas + Columnas

Se pueden combinar .g-col-* y .g-row-* para crear estructuras más complejas.

6 columnas y 3 filas
6x1
HTML

<div class="grid" style="height: 100px;"> 
  <div class="g-col-6 g-row-3 bg-info-neon text-white">6 columnas y 3 filas</div>
  <div class="g-col-6 g-row-1 bg-secondary text-white">6x1</div>
</div>
